# MBTI 深度認知功能自測量表 (Advanced Cognitive Function Assessment)

> [!IMPORTANT]
> **這不是一個普通的心理測驗。**
> 市面上的測驗大多基於「行為表現」與「二分法」（如：你是否遲到？），容易受當下環境與心情影響。
> 本量表基於 **榮格八維認知功能 (Jungian Cognitive Functions)**，旨在挖掘你大腦處理信息與做決定的**底層核心機制**。請誠實面對自己的**本能反應**，而非「工作中的你」或「理想中的你」。

---

## 第一部分：能量與方向 (Energy & Direction)

**Q1. 休息與充電機制**
*當你經歷了高強度的一週工作後，週五晚上你最想做什麼來「恢復血條」？*
*   (A) **獨處**：在安靜的房間裡看書、打遊戲、睡覺，或者做一些不需要與人互動的事情。外部互動會消耗你的電量。
*   (B) **互動**：找朋友聚餐、喝酒、聊八卦，或者去熱鬧的地方感受人氣。獨自一人待太久會讓你感到枯竭或無聊。
    *   *判定：(A) -> I (內傾); (B) -> E (外傾)*

**Q2. 面對未知的態度**
*如果有個突發的、未計劃的長時間旅行邀約，你的第一直覺反應是？*
*   (A) **興奮**：好啊！這充滿了可能性，我不喜歡把行程定得太死，到時候再看情況隨機應變就好。
*   (B) **焦慮**：這會打亂我的安排。我需要先確認住宿、交通和行程表，否則我無法真正放鬆享受。
    *   *判定：(A) -> P (感知); (B) -> J (判斷)*

> [!TIP]
> **初步組合**：根據以上兩題，你大約落在：
> *   **IJ** (深入規劃者) / **IP** (適應性思考者)
> *   **EJ** (果斷行動者) / **EP** (活躍探索者)
> *這只是大方向，接下來的功能對決才是核心！*

## 第二部分：認知功能終極對決 (The Showdown)

請在每一組對決中，選出你**更傾向、更自然**的一種模式（即使你兩種都會）。

### 對決 1：世界觀的構建 (S vs N)

**情境：看到一個蘋果掉在地上。**
*   **Si / Se (實感組)**：
    *   我看到它是紅色的、有點爛了、掉在水泥地上。
    *   我會注意到它的物理細節，或者聯想到過去吃過的某個特定蘋果的味道。
    *   *關注點：現實、當下、細節、過去的經驗驗證。*
*   **Ni / Ne (直覺組)**：
    *   我會想到萬有引力、牛頓、甚至是農業現代化的問題。
    *   我會聯想到這個蘋果象徵著某種衰敗，或者它是如何從樹上掉下來的「可能性」。
    *   *關注點：概念、未來、隱喻、背後的關聯。*

> **你的選擇是？** (S / N)

### 對決 2：決策的依據 (T vs F)

**情境：你的朋友被解僱了，但他工作表現確實很差。他來找你哭訴。**
*   **Ti / Te (思考組)**：
    *   我的第一反應是分析原因：為什麼被解僱？之後該怎麼辦？寫簡歷？提升技能？
    *   我覺得直接安慰解決不了問題，幫他認清事實才是對他好。雖然我可能會試著表現得有同情心，但在心裡我確實在分析。
    *   *關注點：邏輯、效率、事實、解決方案。*
*   **Fi / Fe (情感組)**：
    *   我的第一反應是共情：他現在一定很難過，我得先安撫他的情緒。
    *   我會在意他是否覺得受委屈了，或者團隊氛圍是否不好。至於對錯和下一步，等情緒平復了再說。
    *   *關注點：價值觀、情緒、和諧、人與人的連結。*

> **你的選擇是？** (T / F)

## 第三部分：壓力下的「抓握」反應 (Grip Stress Test)

**這是最精準的驗證方式。** 當你極度壓力、崩潰時，你會變成什麼樣子？這通常揭示了你的**劣勢功能 (Inferior Function)**。

請閱讀以下四種崩潰模式，哪一種讓你感到**最熟悉且羞恥**？

### 模式 A：暴怒的控制狂 (劣勢 Te)
*   平日你可能溫和、隨性。但在極度壓力下，你變得極度挑剔、專斷，對他人的無能感到暴怒。你會突然想要控制一切細節，對別人大吼大叫，覺得「沒人能把事情做好」。
    *   *可能的類型：**INFP, ISFP***

### 模式 B：放縱的享樂者 (劣勢 Se)
*   平日你可能理智、有遠見、自律。但在極度壓力下，你突然想「毀滅」這一切。你會暴飲暴食、瘋狂購物、熬夜打遊戲、看無腦劇，甚至有危險駕駛或性放縱的衝動。你完全沉溺於感官刺激，無法思考未來。
    *   *可能的類型：**INTJ, INFJ***

### 模式 C：情緒化的受害者 (劣勢 Fe)
*   平日你可能邏輯清晰、冷靜、甚至有點冷漠。但在極度壓力下，你突然變得極度敏感、情緒化。你覺得「全世界都針對我」、「根本沒人愛我」。你會突然爆發出無法控制的情緒宣洩，像個孩子一樣無理取鬧。
    *   *可能的類型：**INTP, ISTP***

### 模式 D：災難化的預言家 (劣勢 Ne)
*   平日你可能腳踏實地、注重細節、有條理。但在極度壓力下，你的腦子裡充滿了無數種「最壞的可能性」。你會把一個小錯誤無限放大成災難性的後果（「完了，這次遲到我會被開除，然後破產，最後流落街頭」）。你完全失去了對現實的判斷。
    *   *可能的類型：**ISTJ, ISFJ***

## 第四部分：結果綜合判定 (Self-Type Calculation)

請將你的選擇代入下方：

1.  **I / E** (來自第一部分 Q1)
2.  **S / N** (來自第二部分 對決 1)
3.  **T / F** (來自第二部分 對決 2)
4.  **J / P** (來自第一部分 Q2，但請結合下方的功能驗證)

### 功能驗證表 (Function Stack Check)

找到你最符合的四個字母組合，然後檢查其主導/劣勢功能是否符合你在**第三部分**的選擇。

*   **INFP**: 主導 **Fi** (重視內在價值) / 劣勢 **Te** (壓力下變控制狂) -> *符合模式 A*
*   **ISFP**: 主導 **Fi** (重視內在體驗) / 劣勢 **Te** (壓力下變控制狂) -> *符合模式 A*
    
*   **INTJ**: 主導 **Ni** (遠見) / 劣勢 **Se** (壓力下感官放縱) -> *符合模式 B*
*   **INFJ**: 主導 **Ni** (洞察) / 劣勢 **Se** (壓力下感官放縱) -> *符合模式 B*

*   **INTP**: 主導 **Ti** (邏輯) / 劣勢 **Fe** (壓力下情緒爆發) -> *符合模式 C*
*   **ISTP**: 主導 **Ti** (分析) / 劣勢 **Fe** (壓力下情緒爆發) -> *符合模式 C*

*   **ISTJ**: 主導 **Si** (細節) / 劣勢 **Ne** (壓力下災難化) -> *符合模式 D*
*   **ISFJ**: 主導 **Si** (照顧) / 劣勢 **Ne** (壓力下災難化) -> *符合模式 D*

*(ENxx 與 ESxx 類型的劣勢功能則相反，例如 ENTP 的劣勢是 Si，壓力下會變得過度強迫症、糾結於身體的小病痛或過去的細節)*
